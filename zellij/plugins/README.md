# Zellij Plugins

Download these plugins and place them in this directory:

- **zjstatus** - Custom status bar
  https://github.com/dj95/zjstatus/releases

- **room** - Session/workspace management
  https://github.com/rvcas/room/releases

- **monocle** - Fuzzy finder for tabs/panes
  https://github.com/imsnif/monocle/releases

- **harpoon** - Quick navigation to marked panes
  https://github.com/Nacho114/harpoon/releases

- **zellij_forgot** - Command cheatsheet
  https://github.com/karimould/zellij-forgot/releases
